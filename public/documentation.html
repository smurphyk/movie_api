Murph's Movie API

Documentation

Welcome to my first API! This is a one-stop shop for all you movie junkies out there. With my API, you will be able to access all pertinent information about your favorite movies! First things first, make sure you sign up, update all your personal information, and create a list of your favorite movies!

{ name: 'Action', description: 'Action movies are defined by risk and stakes. To be appropriately categorized inside the action genre, the bulk of the content must be action-oriented, including fight scenes, stunts, car chases, and general danger.', }

Allows a new user to register /users POST A JSON object containing data about new user, including name, username, password, email, and birthday:

{ name: 'Sean Murphy', username: 'PapaSmurph', password: 'password123', email: 'noemail@noemail.com', birthday: '03-23-1990', }

A JSON object containing data about new user and new, unique ID:

{ name: 'Sean Murphy', username: 'PapaSmurph', password: 'password123', email: 'noemail@noemail.com', birthday: '03-23-1990', id: 'ef808594-628e-4ec2-81fe-cb15ac727130' }

Allows user to update personal information /users/[USERNAME] PUT A JSON object containing updated user info, including name, username, password, email, birthday, and ID:

{ name: 'Sean Murphy', username: 'PapaSmurph', password: 'password123', email: 'noemail@noemail.com', birthday: '03-23-1990', id: 'ef808594-628e-4ec2-81fe-cb15ac727130' }

A status message confirming that information was updated:

User [USERNAME] successfully updated their info!

Allows user to add a movie to their list of favorites /users/[USERNAME]/favorites POST A JSON object containing information about movie to add:

A status message confirming that the new movie was added to favorites:

User [USERNAME] successfully added [TITLE] to favorites!
Allows User to removie a movie from favorites, by title /users/[USERNAME]/favorites/[TITLE] DELETE None A status message confirming that the movie was removied from favorites:

Allows users to deregister /users/[USERNAME] DELETE None A status message confirming that the user was deleted:
